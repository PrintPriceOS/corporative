# Standard Sections

## Hero
- headline
- subheadline
- primary CTA
- optional visual

## Workflow
- horizontal or vertical
- max 4-5 steps
- labeled clearly

## Product Grid
- 3-6 cards
- grouped logically

## Value Proposition
- 3-4 benefits
- concise

## Technical Credibility
- architecture
- API
- system reliability

## CTA Banner
- single strong action
