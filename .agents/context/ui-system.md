# PrintPrice Pro UI System

## Visual Identity

- Style: Enterprise SaaS, technical, minimal, structured
- Avoid: startup gradients, playful UI, excessive glassmorphism

## Color System

- Primary: neutral dark or light base
- Accent: ONE controlled accent color only
- Avoid multiple neon accents

## Typography

- Clean sans-serif
- Strong hierarchy
- No decorative fonts

## Layout Principles

- generous whitespace
- clear section separation
- grid-based structure
- max 6-7 sections per page

## Components

- hero
- product cards
- workflow diagram
- stats block
- CTA banner
- footer

## Workflow Rule

If PrintPrice Pro:
- ALWAYS include workflow visualization
- ALWAYS show product relationships
- NEVER isolate products without context

## Product Representation

Products must be grouped as:
- Estimation: Budget
- Validation: Preflight
- Operations: CRM
- Control: Control

## Tone

- technical
- precise
- enterprise
- calm
- confident

## Forbidden Patterns

- random visual styles per page
- inconsistent color systems
- over-designed UI
- marketing fluff layouts
